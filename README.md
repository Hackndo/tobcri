# Tobcri Project
### Description
tobcri is an IRC bot based on Sopel that supports SSL, multi channels, and provides different modules and commands

### Version
0.1

### Requirements

See `requirements.txt` file

### Installation

```sh
mkvirtualenv tobcri -p $(which python3)
python setup.py
```

### Test

```sh
tox
```
